## Required

## Steps

### 1. Clone project

```bash
git clone https://github.com/nertexisdead/testCrud.git

```

### 2. Database settings

```bash
DB_CONNECTION=sqlite
# DB_HOST=127.0.0.1
# DB_PORT=3306
# DB_DATABASE=laravel
# DB_USERNAME=root
# DB_PASSWORD=
```

### 3. Get started

```bash
npm install
composer install
```

### 4. Run

```bash
php artisan serve
```
